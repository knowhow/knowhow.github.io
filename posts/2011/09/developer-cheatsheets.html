developer cheat sheets

September 12 2011, 12:35 AM  by Ernad Husremović

"Cheatsheets" su dokumenti koji daju pregled glavnih komandi određene aplikacije, framework-a ...

Ovo je jedan dobar git cheatsheet. Međutim, ono pored ovakvih, grafički "nalickanih" cheatsheet-ova (koji su naravno super), Chris Wanstrath-ova (alias defunkt) mala ruby aplikacija cheat je genijalna stvar. Jednostavna do boli.

Instalacija:

rvm install 1.9.2 # ako imate instaliran rvm
gem install cheat

Korištenje:

cheat sheets # prikaži sve dostupne cheat sheets 
cheat asciidoc 
cheat ruby 
cheat sed 
cheat chromium
cheat bash
cheat regex | grep -A1 -B5 multi

defunkt je poznato ime u ruby svijetu. Jedan je od vodećih developera u github-u.

Objavio je mali milion opensource projekata. Jedan od najpoznatijih veoma raširenih je resque work queue(*). 

Interesantno je koliko ta nova generacija developera forsira jednostavna rješenja,  rad na konzoli.

Microsoft Visual studio IDE !? Ma bjaaži ... vim, konzola, grep to su za njih glavna ubojita oružja. Prosto im zavidim na njihovoj pameti i kreativnosti.

---

(*) dovoljno je pogledati "watch" i "fork" cifre ovog projekta.
