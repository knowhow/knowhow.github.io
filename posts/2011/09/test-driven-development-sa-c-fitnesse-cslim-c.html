Test driven development sa c++ = Fitnesse + cslim + cppunit

September 21 2011, 1:57 AM  by Ernad Husremović

Object mentori(*) su poznata konsultantska firma iz oblasti software developmenta. Svojevremeno sam pisao članak o prezentaciji Roberta Martina, koji je takođe član "Object Mentor" tima.

Fitnesse je primarno platforma za testiranje java koda, ali su neki pametni ljudi napravili cslim ekstenziju.

Prateći ovaj wiki, podesio sam Fitnesse/cppunit/cslim.

Stvar se začuđujuće brzo i jednostavno podešava.

(*) Oni su, ako sam dobro razumio, jedan od autora popularnog Fitnesse testing framework-a.
