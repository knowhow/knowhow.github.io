"knowhow ERP" i industrijski standardi

October 20 2011, 4:17 AM  by Ernad Husremović

Na svom blogu sam napisao jedan članak o OSS-u i industrijskim standarima. To me je navelo da navedem alate i tehnologije koje koristimo u "knowhow ERP" sa odgovarajućim licencama.

Mobilni klijent knowhow ERP

Razvojna platforma: Appcelerator titanium. Licenca: Apache V2.0

Programski jezik koji koristi application developer: javascript.

Podržane platforme Google Android OS (licenca Apache v2.0), te Apple iOS (*).

Database

a) Database server knowhow ERP

Server: PostgreSQL. Licenca: liberalna BSD/MIT licenca.

Bazne tehnologije: SQL

Database developer, stored procedure language: PL/PGSQL

Podržane platforme: 'đe god ima C-a

b) Mobilni client local database

SQlite, licenca public domain.

Rich client knowhow ERP

Application developer: Qt framework, javascript Qt implementacija - qtscript.

Licenca za Qt framework i alate: LGPL

Core application developer: Qt/C++

xtuple core licenca: xTuple CPAL

Alati za razvoj, dio Qt framework-a:

  1. QtLinguist
  2. QtDesigner
  3. QtCreator

xTuple/knowhow interni report engine: xTuple OpenRPT. Licenca: LGPL

Ostale reporting opcije: 

  • Pentaho BI, Mozila public license 1.1
  • JasperReports BI, GPL

A4 application server

node.js, Licenca: MIT

Ključne komponente: google v8 javascript engine, licenca BSD.

Rezime

Svi alati, od najvišeg do najnižeg nivoa open source software izdati pod "OSI approved" licencama.

---

(*) Postoji i podrška za druge mobilne OS-ove, ali to u ovom trenutku nije predmet interesovanja knowhow ERP pa neću navoditi
