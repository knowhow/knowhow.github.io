Moj prvi github pull request

August 7 2011, 2:48 AM  by Ernad Husremović

Ono što  github čini posebnim jesu njegove "social" osobine. Github je omogućava komunikaciju developera na najprirodniji mogući način - putem izvornog koda(*)
.

"Pull request"(**) je odličan primjer takve komunikacije. Uz pomoć "pull request"-a rezultate svog rada nudimo "upstream" (***) projektu.

Trebao uočiti da je čitav ovaj proces krajnje jednostavan kako za podnosioca, tako i za primaoca zahtjeva.

https://github.com/opscode/cookbooks/pull/185

---
(*) Source code - izvorni kod
(**) Pull request - zahtjev za preuzimanjem koda
(***) Upstream - nadređeni, glavni projekat. U ovom slučaju radi se o glavnom repozitoriju recepata (cookbooks) za chef automatic provision platformu http://www.opscode.com/chef/
